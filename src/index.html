To start chatting, you need to configure an API Key.

  1. Click the Settings button in the top right.
  2. Go to the API tab.
  3. Select your provider (e.g., Google Gemini) and paste your API Key.
  4. Click Save.

Don't have a key? Search for "Gemini API Key" or "Groq API Key" to get one for free.
